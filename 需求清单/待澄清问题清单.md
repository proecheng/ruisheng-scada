#### Q-P07 · TCP 6020 上跑的到底是标准 MBAP+PDU，还是"TCP 上直跑 RTU 帧"？（部分澄清）
- **收件人**：原开发者
- **现已知**：
  - 代码里有标准 CRC16（多项式 0xA001），暗示 **RTU 帧直跑 TCP**
  - 没看到 MBAP Header（Transaction ID / Protocol ID / Length / Unit ID）处理代码
  - TCP 6030 是纯 UTF8 JSON 字符串，不是 ModBus TCP
- **期望回复**：
  - 确认 / 否认 "TCP 6020 直跑 RTU 帧（含 CRC16）"
  - 如果是 MBAP，为什么代码看不到解析？是否走了某个库？

#### Q-P08 · 注册帧 DevSerNumber 字段格式？（部分澄清）
- **收件人**：原开发者 / 设备固件开发者
- **现已知**：数据库层是 `string` 无长度约束（`DataBase.cs:292`）
- **期望回复**：
  - 帧中字段**长度是固定还是可变**？
  - 编码是 ASCII 还是 HEX 还是 BCD？
  - 典型示例（随便给 1 条真实设备的注册帧 HEX）

#### Q-D01 · 历史库 ModBusHis.mdf 为何只有 4MB？
- **收件人**：DBA / 运维
- **背景**：主库 1.27GB，历史库 4MB —— 要么从未启用，要么被清空
- **期望回复**：
  - 历史库是否曾投产？投产时间段？
  - 如果清空过，为什么？
  - **是否有过去 1 年的真实历史数据？在哪？**
  - 如果没有历史数据，新系统是否需要补回测试数据？

#### Q-V01 · 现网真实规模？
- **收件人**：项目经理 / 运维
- **期望回复**（给数字即可）：
  - 在册设备数 / 当前在线设备数
  - 活跃用户数（近 30 天登录）
  - 日均上报报文量（条/天）

- **收件人**：DBA / 运维 / 原开发者
- **背景**：电能终端波形分析工具代码 `DataBase.cs:159-165` 里有 3 个 UDP 端口常量：
